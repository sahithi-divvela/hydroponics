# 🌱 Smart Hydroponics System

An AI-driven Smart Hydroponics system designed to automate plant growth monitoring and control environmental conditions efficiently.

## 🚀 Project Overview
This project focuses on building an intelligent hydroponics setup that monitors and controls:

- pH levels
- TDS (Total Dissolved Solids)
- Water level
- Temperature
- Humidity
- Light intensity

The system aims to improve plant growth, reduce water usage, and minimize manual intervention.

---

## 🛠️ Technologies Used
- Python
- Arduino
- Embedded C
- Sensors (pH, TDS, DHT11, Water Level)
- IoT Integration
- Data Logging & Monitoring

---

## ⚙️ Features
- Real-time monitoring of nutrient levels
- Automated control of pumps and lights
- Data visualization
- Alert system for abnormal readings
- Scalable for closed-loop and open-loop environments

---

## 📊 System Architecture
Microcontroller collects sensor data → Data processed using Python → Actuators controlled based on threshold logic → User monitoring dashboard
